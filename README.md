# Open Source Engineering Trend Observatory

A live data engineering and analytics platform that processes global GitHub activity and generates insights about engineering trends.

## Stack

Python  
Spark  
SQL  
DuckDB  
Docker  
GitHub Actions  
AWS S3  

## Goal

Build a production-style data pipeline that:

- ingests GitHub Archive data
- processes events with Spark
- computes analytics metrics
- generates charts
- publishes outputs for a live portfolio dashboard
